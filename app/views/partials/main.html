PNG STEGANOGRAPHY

This application allows embedding a file into a png or bmp image, returning a pair of png and bmp files, and extracting embedded files from them. Notice that, you will not receive bmp result if the input is a paletted png.

You can drop files into the boxes or click on the boxes to select files. By choosing an image file on the left box of the EMBED section, you will see the maximum capacity for embedding file on the box beside. Under the boxes are options including password and algorithm. Password is optional. When you are ready, push the button on the right hand side (Embed or Extract) to make the request.

All requests will be displayed as a list of waiting processes. You may make more requests while waiting for results.
