PNG STEGANOGRAPHY

This application allows embedding a file into a png or bmp image, returning a pair of png and bmp files, and extracting embedded files from them. Notice that, the bmp file received when the input is a paletted png cannot be used to extract information, since the palette has been omitted.

You can drop files into the boxes or click on the boxes to select files. By choosing an image file on the left box of the EMBED section, you will see the maximum capacity for embedding file on the box beside. Under the boxes are options including password and algorithm. Password is optional. The request will be made automatically when all required files are selected. If you want to request with a new password, push the button on the right hand side (Embed or Extract) to make a new request.

All requests will be displayed as a list of waiting processes. You may make more requests while waiting for results.
